Para rodar o projeto é preciso baixar as dependencias do node dentro de 3 pastas atraves do terminal

1.Dentro da pasta base-backend, abra o terminal e digite o comado "npm install", para instalar as dependencias do node, em seguida o comando "npm run serve", para inicializar o localhost na porta 3000, para realizar os testes o insomnia sera utilizado

Dentro da pasta Front existem duas pastas que precisam de comandos no terminal para funcionarem :

2. Entre na pasta "BackUsuarios", depois entre na pasta "node-api-discover-main" e abra o terminal dentro dela, os seguintes comandos devem ser executados, "mpm install" e logo em seguida "npm run start" para inicializar o servidor
3. Entre na pasta "BackViagens", depois entre na pasta "node-api-discover-main2" e abra o terminal dentro dela, os seguintes comandos devem ser executados, "mpm install" e logo em seguida "npm run start" para inicializar o servidor

Logo apos fazer estes passos a aplicação esta habita a ser utilizada no navegador, pelo localhost (http://127.0.0.1:5500/Desafio.html)
